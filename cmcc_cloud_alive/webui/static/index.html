使用说明

在同一页完成账号配置与保活控制，无需向导分步。

  • 上方 Composer 填账号密码等，主按钮「保存并保活」。
  • 下方卡片可并列管理多个账号：开始 / 停止 / 配置等；删除账号在配置弹窗内。
  • 关掉浏览器页面后，已启动的保活仍在后台运行。
  • 客户端类型可选 linux / windows / mac（保存后会写进档案，不会再自动变回 linux）。
  • 协议选中兴（ZTE）或深信服（SCG），按你的云电脑实际类型选择。
  • 运行日志支持双击查看全部。

访问密钥

可选。保护控制台 API，不影响保活。打开原网址即可，密钥在门页填写，不要拼进 URL。

  • 登录门输入；改密 / 清本机用顶栏「设置令牌」。
  • 也可 .env 写 CMCC_WEBUI_TOKEN（已有 token 文件优先）。

细节见 README「WebUI 访问密钥」。
