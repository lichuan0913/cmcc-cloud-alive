使用说明

在同一页完成账号配置与保活控制，无需向导分步。

  • 上方 Composer 填账号密码等，主按钮「保存并保活」。
  • 下方卡片可并列管理多个账号：开始 / 停止 / 配置 / 桌面登出；删除账号在配置弹窗内。
  • 关掉浏览器页面后，已启动的保活仍在后台运行。
  • 客户端类型一般保持 Linux；协议选中兴或深信服。
